Binomial Distribution

Definition

The binomial distribution models the number of successes in a fixed number of independent Bernoulli trials, each with the same probability of success.

Probability Mass Function (PMF)

P(k) = C(n, k) · pk · qn−k

Where q = 1 − p and C(n, k) is the binomial coefficient.

Mean

E(X) = n · p

Variance

Var(X) = n · p · q

Distribution Formula

(p + q)n

Tips:
  • Use when each trial has only two outcomes: success or failure.
  • The total number of trials (n) must be fixed.
  • Trials must be independent, and probability (p) stays constant.
  • Mean and variance help summarize the expected behavior.
